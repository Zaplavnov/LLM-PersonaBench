## PAPI
**PAPI-300K** — это набор из 300K датасетов для PAPI, включающий вопросники IPIP-NEO-120 и IPIP-NEO-300 с ответами 300K участников

**Ссылка для скачивания**: [Google Drive](https://drive.google.com/file/d/1V0JkJL3KMn6ZpOqOFVkfbLZD1HqQJgS5/view)

## PAPA
**PAPA-120-600K** — это набор из 600K датасетов для PAPA, включающий только вопросник IPIP-NEO-120 с ответами 600K участников.

**Ссылка для скачивания**: [Google Drive](https://drive.google.com/file/d/1KRhpTCwSMS47GYnmHwYRPnmxF6FOGYТф/view)


## Kaggle Datasets
Датасеты личностных опросников IPIP-NEO с Kaggle:

### IPIP-NEO-120
**IPIP-NEO Big Five Personality 120-Item Version** — датасет с результатами IPIP-NEO-120 вопросника для оценки пяти черт личности и аспектов

- **Файл**: `IPIP120-SCORES.csv`
- **Содержит**: Уже рассчитанные скоры по аспектам и по основным пяти чертам личности
- **Ссылка**: [Kaggle Dataset](https://www.kaggle.com/datasets/edersoncorbari/pip-neo-big-five-personality-120-item-version?resource=download)

### IPIP-NEO-300
**IPIP-NEO Big Five Personality 300-Item Version** — полный датасет с результатами IPIP-NEO-300 вопросника для более детальной оценки личностных черт

- **Файл**: `IPIP300-SCORES.csv`
- **Содержит**: Уже рассчитанные скоры по аспектам и по основным пяти чертам личности
- **Ссылка**: [Kaggle Dataset](https://www.kaggle.com/datasets/edersoncorbari/ipip-neo-big-five-personality-300-item-version/data)
